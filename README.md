# Performance-Report

### FLOW PROJECT

1. User register (username + password)
2. User isi data diri
3. Admin verifikasi data officer
4. Admin assign data performance sesuai data yang telah diinput
5. Officer lihat visualisasi
6. AI summary dibuat
7. Officer download hasil

### Saran Visualisasi

| Data                 | Visual               |
| -------------------- | -------------------- |
| Trend performa       | Line chart           |
| Perbandingan kuartal | Bar chart            |
| Komposisi KPI        | Radar / Spider chart |
| Status KPI           | Progress bar         |
| Summary cepat        | KPI Cards            |
